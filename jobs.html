System Administrator

Position Reference: SA123

Company: Byte Me Sdn Bhd.

Salary Range: RM 6000 - RM 8000

Reports To: IT Manager

Overview Requirement Apply Now
×

Position Overview

The System Administrator will be responsible for managing and maintaining the company’s server infrastructure, ensuring optimal performance, security, and stability across all systems.

Key Responsibilities

  • Manage and monitor all installed systems and infrastructure.
  • Install, configure, test, and maintain operating systems, application software, and system management tools.
  • Ensure security through access controls, backups, and firewalls.
  • Troubleshoot hardware and software issues as they arise.
  • Monitor system performance and ensure reliability and availability.
  • Perform regular backup operations and implement appropriate processes for data protection and disaster recovery.
×

Required Qualifications, Skills, and Knowledge


Essential

  1. Bachelor’s degree in Computer Science or a related field.
  2. At least 3 years of experience in system administration.
  3. Proficiency in Linux and Windows server environments.
  4. Experience with virtualization (e.g., VMware or Hyper-V).
  5. Strong understanding of network infrastructure and security.

Preferable

  • Experience with cloud platforms (AWS, Azure).
  • Certifications such as LFCSA, RHCSA, CompTIA, MCSE, or AWS Certified Solutions Architect.
  • Knowledge of scripting and programming languages such as Bash,Python, C#.

Position Overview

The DevOps Engineer will work closely with software developers, system operators, and other IT staff to oversee code releases and deployments, ensuring automation and continuous integration within the organization’s infrastructure.

Key Responsibilities

  • Implement and maintain CI/CD pipelines for code deployment.
  • Automate infrastructure provisioning and scaling using tools like Terraform.
  • Monitor system performance and security, applying patches and updates as necessary.
  • Collaborate with software development teams to streamline development processes.
  • Deploy, monitor, and troubleshoot containerized applications using Docker and Kubernetes.
×

Required Qualifications, Skills, and Knowledge


Essential

  1. Bachelor’s degree in Computer Science, IT, or a related field.
  2. 5+ years of experience in a DevOps or related role.
  3. Strong experience with cloud services (AWS, GCP, or Azure).
  4. Proficiency with CI/CD tools such as Jenkins, GitLab, or CircleCI.
  5. Experience with containerization (Docker) and orchestration (Kubernetes).

Preferable

  • Knowledge of Infrastructure as Code (IaC) tools such as Terraform or CloudFormation.

  • Familiarity with scripting and programming languages like Python, Ruby, or Shell.
